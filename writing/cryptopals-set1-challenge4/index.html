Cryptopals - Set 1 Challenge 4: Detect single-character XOR

February 23, 2022 | Suggest Changes

This post provides an explanation for the fourth cryptopals challenge, how to detect if a string was encrypted with single-byte XOR and decrypt it.

Detect single-character XOR#

From Cryptopals Set 1 Challenge 4:

One of the 60-character strings in this file has been encrypted by single-character XOR.

Find it.

(Your code from #3 should help.)

Solution#

Solving this challenge is very similar to Set 1 Challenge 3.

Steps to solve this challenge:

  1. Read the file.
  2. Run the DecryptSingleByteXOR function from last challenge on each line of the file.
  3. The line with the highest score is the answer.
package main

import (
	"encoding/hex"
	"fmt"
	"io/ioutil"
	"log"
	"net/http"
	"strings"
)

func main() {
    // Read the input from https://cryptopals.com/static/challenge-data/4.txt.
    req, err := http.Get("https://cryptopals.com/static/challenge-data/4.txt")
    if err != nil {
    	log.Fatal(err)
    }
    defer req.Body.Close()

    body, err := ioutil.ReadAll(req.Body)
    if err != nil {
    	log.Fatal(err)
    }

    // Read through each line of the file, and compute the highest possible
    // score from single-byte XOR decryption.
    //
    // The line with the highest score is the answer.
    var (
    	output       string
    	highest int
    )
    lines := strings.Split(string(body), "\n")
    for _, line := range lines {
    	out, s, err := set1.DecryptSingleByteXOR(line)
    	if err != nil {
    		log.Fatal(err)
    	}
    	if s > highest {
    		highest = s
    		output = out
    	}
    }
    fmt.Println(string(output))
}
